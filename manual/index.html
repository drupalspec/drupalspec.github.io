Правила работы

SceneBuilder используется только как просмотрщик. Все изменения в код должны вводиться руками.

Надо максимально избегать стилей в тексте: maxWidth, prefHeight, minWidth, minHeight, <Insets> и прочая ерунда запрещены.

При переверстке текущих шаблонов надо особенно аккуратными быть с параметрами text, fx:define, fx:id и id. Не копируйте бездумно из примеров, а обязательно заменяйте на текущие значения.

Внутренний CSS для JavaFX — modena.css, caspian.css

Основной файл визарда

FXML

    <SplitPane id="splitPane" dividerPositions="0.24"
           stylesheets="@../../../common/css/metro-ui.css"
           styleClass="wt-scene"
           xmlns="http://javafx.com/javafx/8" xmlns:fx="http://javafx.com/fxml/1">
        <items>
          <VBox styleClass="wt-sidebar">
             <Label styleClass="wt-sidebar__header" fx:id="mainTitle"/>
             <VBox fx:id="navigationPane" styleClass="wt-sidebar__menu" />
          </VBox>
          <AnchorPane styleClass="wt-content" fx:id="contentAnchorPane">
                <ScrollPane styleClass="wtb-frame" AnchorPane.leftAnchor="0.0">
                    <Pane styleClass="wt-iframe" fx:id="wizardStepsPane"/>
                </ScrollPane>
          </AnchorPane>
        </items>
    </SplitPane>

Мы используем SplitPane.wt-scene с подключенным файлом стилей metro-ui.css.

Сайдбар слева реализуется через VBox.wtb-sidebar. В нем находятся Label.wt-sidebar__header и VBox.wt-sidebar__menu

Меню

Меню генерируется автоматически, а для оформления использует классы .wt-sidebar__link и .wt-sidebar__sublink.

Файлы форм визарда

Используйте VBox, как форму, и HBox, Grid и прочие Pane, как строки формы.

Для VBox указывайте класс .wt-form, который дает отступы справа и слева. Ширину и высоту формы указывать не нужно.

FXML

    <VBox id="MainPane" fx:id="mainAnchorPane" styleClass="wt-form"
      stylesheets="@../../../common/css/metro-ui.css"
      xmlns="http://javafx.com/javafx/8" xmlns:fx="http://javafx.com/fxml/1">
          <children>
          </children>
    </VBox>
Секция формы

Секции формы формируются из HBox, Grid и прочих Pane с классами из коллекции .wt-row c элементами __element и, опционально, модификаторами _modificator. Все секции имеют заранее заданные вертикальные отступы между друг другом.

FXML

    <HBox styleClass="wt-row__element_modificator">
        <children>
        </children>
    </HBox>

Ширина внутреннего пространства секции (т.е. с вычетом отступов) равна 880px.

Если после лейбла необходим знак вопроса с подсказкой, используйте компонент подсказки. У него уже есть отступ слева от предыдущего элемента.

FXML

    <Pane fx:id="helper" />

После кажого лейбла со знаком вопроса или без надо размещать спейсер.

FXML

    <Pane HBox.hgrow="ALWAYS"/>
