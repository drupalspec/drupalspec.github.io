Description

Interrupt KITAS sensor cable from KITAS. Connect the “WorkshopTab Cable KITAS direct” with the sensor. Click on Read KITAS S/N and compare this with the Technical printout of the DTCO. The serial number must be identical with the one on the first print out.
If there is a difference please detect the cause and re-establish the original conditions. Stop the calibration if the original condition could not be re-established. Remark this anyway in the Inspection report.
Intrrupt the DTCO power supply (connector A)
Remove the test connection and connect the KITAS 2171 with the original KITAS cable.Connect power supply to DTCO (connector A) again.
The DTCO re-boots. Wait till the plugged in workshop card is re-autentificated. Error message 31/19 appears in the DTCO display. Please confirm.

Remount and lock the DTCO again.

Close

Description

Interrupt KITAS sensor cable (connector B) from DTCO®. Replace origininal vehicle KITAS cable by a separate test connection (external KITAS sensor cable). Connect the cable.
Connect power supply to DTCO (connector A) again.
The DTCO re-boots. Wait till the plugged in workshop card is re-autentificated. Error message 31/19 appears in the DTCO display. Please confirm.
Then start manually KITASactivation and wait till the activation process has finished.
Create printout “Technical Data” (2)
The printout may be interrupted after the appearance of the sensor serial number.
The serial number must be identical with the one on the first print out.
If there is a difference please detect the cause and re-establish the original conditions. Stop the calibration if the original condition could not be re-established. Remark this anyway in the Inspection report.
Interrupt the DTCO power supply (connector A)
Remove the test connection and connect the KITAS 2171 with the original KITAS cable.
Connect power supply to DTCO (connector A) again.
The DTCO re-boots. Wait till the plugged in workshop card is re-autentificated. Error message 31/19 appears in the DTCO display. Please confirm.

Remount and lock the DTCO again.

Close
