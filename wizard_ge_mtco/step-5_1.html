MTCO Parameterisation, Adaption and Tolerance Check

Check the following parameter and actualise if required: odometer value, VIN, installation date, date (next) calibration, TCO parameter n-Constant, TCO parameter drive shaft I/U, TCO parameter configuration, TCO parameter sealing number, TCO parameter serial number
?
At this point it’s depending of the measurement method
Street: - Measure dynamic e.g with the Tyre Meter and program tire circumference (l figures)
- Measure and program vehicle constant (w figures).
  • Important hint: program w-value always with the next step.

Bench: - Measure dynamic e.g with the Tyre Meter and program tire circumference (l figures)
- Vehicle stand on the roller, let roller run
- Start the w/l Calibration and follow the instruction the menue
- When the measurement is finished, roller stop
- Watch display, the WorkshopTab shows the determined correction value, please note to fill in these later in the Report
- Adapt w-k
  • With the adaption the l-value is also adopted in the DTCO
If required, the vehicle can leave the roller bench again
?
Measure the accuracy of the odometer (device tolerance test). Max. failure +/- 1 %
Check the time deviation of the clock. Max. failure +/- 2 minutes per day or +/- 10 minutes in 7 days.
Measure the accuracy of the speed display (device tolerance test) with the function “Variable Speed”. Measurement points: 20, 80 and 180 km/h. Max. failure +/- 3 km.
Test chart: Prepare and insert the tachograph charts (Attention to measuring range!)
Start the test chart
Evaluate the test chart

Before starting the test drive, delete the error memory!
Drive a test track with a distance if min. 1 km with a inlaid TCO chart
  • Check the uniform display of the cluster
  • Monitor the display of the device
  • Check the additional functions
Finish the test track
  • Evaluate records
Diplay the error memory, there must be no new error in memory

Sealing the test- & program interface
Check the sealing of the KITAS if necessary new seal.
